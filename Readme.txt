	Description:
	A supreme commander 2 mods that rebalance all costs, and power units
	
	This mode is more realistic and render the game more interesting.

	To build it :
	create a directory z_balanced that contains my 2 repository directory (mods & units)

	zip it & rename the .zip as .scd:
	z_balanced.scd

	copy the z_balanced.scd file into the gamedata\ supreme commander2 directory

	example for steam users:
	E:\Steam\SteamApps\common\Supreme Commander 2\gamedata
